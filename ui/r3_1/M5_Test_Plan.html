Content Assist (Dani, Tobias)

  • check enhanced for-loop template
  • check completion of enum types and constants
  • Check arguments insertion when completing a generic type
    • enable Java>Editor>Code Assist, Advanced>'Fill argument names'
  • Check parameter guessing when completeing a method of a generic type with a wildcard type bound
    • enable Java>Editor>Code Assist, Advanced>'Guess filled argument names'
    • example: invoke code assist at the caret (|) location:
      		 void method(List<? extends Number> upper, List<? super Number> lower) {
		 		 upper.|
		 		 lower.|
		 }
  • Check parameter guessing and autoboxing
    • enable Java>Editor>Code Assist, Advanced>'Guess filled argument names'
    • example: complete the call to get(int) 
      		 void method(Integer i1, int i2, List list) {
		 		 list.get|
		 }
  • Check content assist for type variables within @param javadoc tags
