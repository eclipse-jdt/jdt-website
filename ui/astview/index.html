org.eclipse.jdt.astview - AST View
Purpose

A view to visualize the AST (abstract syntax tree) of a Java file open in the editor. Navigate from text selection to AST nodes and from nodes to selections.
Install

Eclipse 3.0

  • requires a SDK build > I20040309
  • manually download the plugin org.eclipse.jdt.astview and extract the plugins into the plugins directory of your Eclipse install.
      or
  • use the update feature and go to the AST View Update Site:
        http://dev.eclipse.org/viewcvs/index.cgi/%7Echeckout%7E/jdt-ui-home/plugins/org.eclipse.jdt.astview/AST-View-Site
Usage
  1. Open the AST view from the view menu: Window > Show View > Other... Java > AST View
  2. Open a Java file in the editor
  3. Click on 'Show AST of active editor' ( ) to populate the view: The view shows the AST of the file opened in the editor and will also reveal the element corresponding to the current text selection
  4. Enable 'Link with editor' () to automatically track the active editor and the selection in the active editor.
  5. Double click on AST nodes to get the corresponding element revealed in the editor.
  6. Double click again to see the 'extended range' of a node, that means the range including all comments associated with it (comment mapper heuristic).
  7. When the AST's underlying document has been changed use 'Refresh' ()  to update the AST.
AST View Elements

There are three kinds of elements shown in the AST view

  • AST nodes are shown in gray color and with source range information. Children of AST nodes are the node properties.
  • node properties have names that are all all capitalized. Node properties represent lists of child nodes, a child node or a primitive node attribute (String, int or boolean)
  • bindings are shown in brackets. Only selected bindings are shown,
Advanced Features
  • You can toggle between the AST Level 2 and AST level 3 in the view menu (triangle). AST level 3 is the extended and modified AST that include the new 1.5 language constructs. In I20040309, the level 3 AST is not yet available.
