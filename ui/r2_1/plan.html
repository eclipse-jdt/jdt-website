Overall 2.1 Plan
The 2.1 plan below is a proto plan and input from the community is very welcome.

The overall themes for JUI are:

  • reduce gestures, improve navigation
  • more refactoring support
  • smarter editing

Planned Items

    • address not present exception issue

    • push some JDT independent editor function into Platform Text component

    • investigate start-up performance

  • Improve test coverage
  • General UI issues
    • hierarchical package browsing
