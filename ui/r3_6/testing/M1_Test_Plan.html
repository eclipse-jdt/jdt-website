Export All... from Formatter Preference page (Markus, Dani - FAQ: Raksha)

  • test that it exports all the non-default profiles into an xml file
  • test that it reports duplicate file name, profile names
  • test that 'Import...' can import multiple profiles

Open Implementation command in Navigate menu (Markus, Dani - FAQ: Raksha)

  • test that it works only on overridable methods
  • test that it works for method declaration, invocation, super method invocation
  • test that it is disabled for structured selection (from other views: Outline view, Call Hierarchy view)

Emulated resize support for hovers on Linux (Dani, Markus - FAQ: Markus)

  • test that hovers based on AbstractInformationControl are resizable (see bug 62652)
  • Note: Source hovers are not yet resizable, but content assist additional info popups are
  • test that the resize support "feels" right (wrong cursor, bad corner cases, etc.)

Javadoc without browser widget (Dani, Markus - FAQ: Markus)
